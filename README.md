# Sightseeing Roma

**Sightseeing Roma** is a web application that provides a comprehensive platform for booking bus services for sightseeing tours in Rome. The application is designed to offer a seamless user experience, making it easy to explore and book sightseeing services.

## Live Site
Access the live site here: [Sightseeing Roma](https://resonant-starship-74097e.netlify.app/)

## Features
- **Browse Tours:** Explore various sightseeing bus tours available in Rome.
- **Responsive Design:** The website is fully responsive, ensuring a great experience on any device.
- **Easy Booking:** User-friendly interface for quick and hassle-free bookings.
- **Interactive UI:** Engaging design to highlight tours and services effectively.

## Technologies Used
- **Frontend:** React.js, HTML5, CSS3
- **Styling:** Tailwind CSS (if applicable, specify)
- **Hosting:** Netlify

## Installation
If you want to run the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/sightseeing-roma.git
   ```

2. Navigate to the project directory:
   ```bash
   cd sightseeing-roma
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open the application in your browser:
   ```
   http://localhost:3000
   ```

## Deployment
The project is deployed using **Netlify**. To redeploy or make changes:

1. Build the project:
   ```bash
   npm run build
   ```
2. Deploy the `build/` folder to Netlify.
